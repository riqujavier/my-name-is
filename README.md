# Javier Riquelme - Frontend Developer Portfolio

Welcome to the source code of my personal portfolio! This application is designed to showcase my journey as a Frontend Developer, highlighting my experience with modern web technologies, React, and Component-based architecture.

## ✨ Features
- **Modern UI/UX:** Built with a stunning dark "glassmorphism" aesthetic and interactive hover states.
- **Bilingual Interface:** Fully functional English and Spanish localization handled via a bespoke React Context provider.
- **Responsive Design:** Optimized for a flawless viewing experience across mobile, tablet, and desktop devices.
- **Secure & Confidential:** Real-world client projects are clearly labeled to respect NDAs while still demonstrating technical capability.
- **SEO Optimized:** Built with semantic HTML and comprehensive Open Graph meta tags for rich social sharing.

## 🛠 Tech Stack
- **Framework:** [Next.js](https://nextjs.org/) (React)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Language:** TypeScript & modern JavaScript
- **CMS Experience:** [Magnolia CMS](https://www.magnolia-cms.com/)

## 🚀 Getting Started Locally

1. Clone the repository and install the dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to view the application locally.
